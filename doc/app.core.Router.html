Name Type Attributes Description
opt_noFragment boolean <optional>
 set to true to hide fragment when using HTML5 history.
opt_blankPage goog.html.TrustedResourceUrl <optional>
 url to a blank page - needed if HTML5 is not available and you don't want to show the fragment.
opt_input HTMLInputElement <optional>
 The hidden input element to be used to store the history token. If not provided, a hidden input element will be created using document.write.
opt_iframe HTMLIFrameElement <optional>
 The hidden iframe that will be used by IE for pushing history state changes, or by all browsers if opt_noFragment is true. If not provided, a hidden iframe element will be created using document.write.
